Синтаксис определения метода

Метод в C# - это блок кода, который выполняет определенную задачу. Он может принимать параметры и возвращать значение.


[модификаторы] возвращаемый_тип ИмяМетода([параметры])
{
    // Тело метода
}

Где:

  • модификаторы - это модификаторы доступа и другие ключевые слова (например, public, private).
  • возвращаемый_тип - это тип данных, который метод вернет после выполнения.
  • ИмяМетода - это идентификатор метода.
  • параметры (необязательно) - это значения, которые передаются методу для выполнения операций.

Пример:


public int Add(int a, int b)
{
    return a + b;
}

Этот метод Add принимает два целых числа a и b и возвращает их сумму.

Параметры методов

Методы могут иметь параметры, которые представляют значения, передаваемые методу для выполнения операций.

Синтаксис определения метода с параметрами:


[модификаторы] возвращаемый_тип ИмяМетода(тип_параметра1 ИмяПараметра1, тип_параметра2 ИмяПараметра2, ...)
{
    // Тело метода
}

Способы передачи параметров

Параметры – значения

Параметры передаются в метод по значению. В методе создается копия переданного значения, и изменения, сделанные внутри метода, не влияют на оригинальное значение.


public void ModifyValue(int x)
{
    x = x * 2;
}

Параметры – ссылки

Параметры передаются в метод по ссылке, что позволяет изменять оригинальное значение из метода.


public void ModifyValue(ref int x)
{
    x = x * 2;
}

Выходной параметр

Метод может иметь выходной параметр, который используется для возврата значения из метода.


public void GetValues(out int x, out int y)
{
    x = 10;
    y = 20;
}

Примеры методов

Давайте рассмотрим несколько примеров методов:

Пример 1: Метод с одним параметром


public void Greet(string name)
{
    Console.WriteLine($"Привет, {name}!");
}

Этот метод Greet принимает один параметр name, представляющий имя человека, и выводит приветствие на консоль.

Пример 2: Метод с возвращаемым значением


public int Square(int num)
{
    return num * num;
}

Этот метод Square принимает один параметр num, возводит его в квадрат и возвращает результат.
