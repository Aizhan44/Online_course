Ввод-вывод элементов одномерного массива

Для ввода-вывода элементов одномерного массива в C# используются циклы и стандартные методы языка.

Пример вывода элементов массива:


using System;

class Program
{
    static void Main()
    {
        // Создаем и инициализируем массив
        int[] numbers = { 1, 2, 3, 4, 5 };

        // Выводим элементы массива на экран
        foreach (int num in numbers)
        {
            Console.WriteLine(num);
        }
    }
}

Пример ввода элементов массива:


using System;

class Program
{
    static void Main()
    {
        // Создаем массив
        int[] numbers = new int[5];

        // Заполняем массив данными, введенными пользователем
        for (int i = 0; i < numbers.Length; i++)
        {
            Console.Write($"Введите число {i + 1}: ");
            numbers[i] = int.Parse(Console.ReadLine());
        }

        // Выводим элементы массива на экран
        foreach (int num in numbers)
        {
            Console.WriteLine(num);
        }
    }
}

Методы класса Array

Класс Array в C# предоставляет множество полезных методов для работы с массивами.

  • Length: Возвращает длину массива.
  • Copy: Создает поверхностную копию массива.
  • CopyTo: Копирует элементы массива в другой массив.
  • IndexOf: Возвращает индекс первого вхождения указанного элемента в массиве.
  • LastIndexOf: Возвращает индекс последнего вхождения указанного элемента в массиве.
  • Reverse: Изменяет порядок элементов массива на обратный.

Примеры использования методов:


using System;

class Program
{
    static void Main()
    {
        // Примеры методов класса Array
        int[] numbers = { 1, 2, 3, 4, 5 };
        Console.WriteLine($"Длина массива: {numbers.Length}");

        // Создание копии массива
        int[] copy = (int[])numbers.Clone();
        Console.WriteLine($"Копия массива: {string.Join(", ", copy)}");

        // Копирование элементов массива в другой массив
        int[] destination = new int[3];
        numbers.CopyTo(destination, 0);
        Console.WriteLine($"Копия элементов в новый массив: {string.Join(", ", destination)}");

        // Индекс первого вхождения элемента
        int index = Array.IndexOf(numbers, 3);
        Console.WriteLine($"Индекс первого вхождения числа 3: {index}");

        // Индекс последнего вхождения элемента
        int lastIndex = Array.LastIndexOf(numbers, 3);
        Console.WriteLine($"Индекс последнего вхождения числа 3: {lastIndex}");

        // Изменение порядка элементов на обратный
        Array.Reverse(numbers);
        Console.WriteLine($"Обратный порядок элементов: {string.Join(", ", numbers)}");
    }
}

Пример использования метода Sort():


using System;

class Program
{
    static void Main()
    {
        // Создаем и инициализируем массив
        int[] numbers = { 5, 3, 2, 4, 1 };

        // Сортируем массив
        Array.Sort(numbers);

        // Выводим отсортированные элементы массива на экран
        foreach (int num in numbers)
        {
            Console.WriteLine(num);
        }
    }
}
