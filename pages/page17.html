Задания по вариантам

 

Тема 1. Операторы и выражения

 

Задание 1. Напишите программу на C#, которая вычисляет сумму двух чисел и отображает результат.

Задание 2. Напишите программу на C#, которая определяет, является ли входное число четным или нечетным.

Задание 3. Напишите программу C#, которая проверяет, является ли входная строка палиндромом.

Задание 4. Напишите программу на C#, вычисляющую факториал входного числа.

Задание 5. C#, определяющее, является ли входное число простым числом. написать программу.

Задание 6. Напишите программу на C#, которая находит корень квадратного уравнения.

Задание 7. Напишите программу на C#, вычисляющую сумму цифр числа.

Задание 8. Напишите программу C#, которая находит наибольший общий делитель двух чисел.

Задание 9. Напишите программу C#, которая сортирует массив целых чисел.

Задание 10: Напишите программу на C# для нахождения среднего арифметического массива чисел.

 

Тема 2: Циклы и условия

 

Задание 1: Напишите программу на C# для отображения на экране всех чисел от 1 до 10.

Задача 2. Напишите программу на C#, которая запрашивает у пользователя число и отображает таблицу умножения для этого числа.

Задание 3. Напишите программу на C#, которая отображает сумму всех четных чисел от 1 до 100.

Задание 4. Напишите программу на C# для нахождения суммы всех чисел от 1 до 100, кратных 3 или 5. напишите.

Задание 5. Напишите программу C#, которая проверяет, является ли входное число простым.

Задание  6. Нахождение наименьшего общего делителя двух чисел на C# напишите программу.

Задание 7. Напишите программу на C#, отображающую все буквы английского алфавита.

Задание 8. Напишите программу на C#, отображающую таблицу умножения от 1 до 10.

Задание 9. Напишите программу на C#, которая отображает факториал входного числа.

Задание 10. Напишите программу на C#, которая отображает сумму цифр числа.

 

Тема 3: Функции и методы

 

Задание 1. Напишите функцию на C#, которая принимает два целых числа и возвращает их сумму.

Задача 2. Напишите метод на C#, который принимает строку в качестве аргумента и выводит ее на экран.

Задание 3. Напишите функцию на C#, которая принимает массив целых чисел и возвращает среднее значение элементов этого массива. напишите.

Задание 4. Напишите метод на C#, который принимает два числа в качестве аргументов и возвращает результат их умножения.

Задание 5. Напишите функцию на C#, которая принимает строку и символ в качестве аргументов и возвращает количество вхождений этого символа в строку. напишите.

Задание 6. Напишите метод на C#, который принимает массив строк и печатает каждый элемент этого массива в новую строку.

Задание 7: В C# принимает два числа и возвращает true, если первое число делится на второе без остатка, в противном случае — false. напишите функцию, которая это сделает.

Задание 8. Напишите метод на C#, который принимает массив чисел и возвращает наибольшее значение в этом массиве.

Задание 9. Напишите функцию на C#, которая принимает строку и возвращает ее длину.

Задание 10: Напишите метод на C#, который принимает два числа и возвращает их разницу.
