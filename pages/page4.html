Операторы присваивания

Операторы присваивания в языке C# используются для присваивания значения переменной. Они позволяют изменять значения переменных во время выполнения программы.

Примеры операторов присваивания:


int a = 10;
int b = a;

Операторы ветвления (if-else, switch)

Операторы ветвления позволяют программе принимать решения и выполнять различные действия в зависимости от условий.

Оператор if-else

Оператор if-else позволяет выполнить блок кода, если указанное условие истинно, иначе выполнить другой блок кода.


int x = 10;
if (x > 0)
{
    Console.WriteLine("Число положительное");
}
else if (x < 0)
{
    Console.WriteLine("Число отрицательное");
}
else
{
    Console.WriteLine("Число равно нулю");
}

Оператор switch

Оператор switch позволяет выбрать из множества возможных вариантов и выполнить соответствующий блок кода.


int day = 3;
switch (day)
{
    case 1:
        Console.WriteLine("Понедельник");
        break;
    case 2:
        Console.WriteLine("Вторник");
        break;
    default:
        Console.WriteLine("День недели неизвестен");
        break;
}

Оператор goto

Оператор goto в языке C# позволяет переходить к определенным меткам в коде. Однако его использование не рекомендуется, так как это может привести к усложнению кода и ухудшению его читаемости.

Пример использования оператора goto:


int i = 0;
start:
i++;
Console.WriteLine(i);
if (i < 10)
    goto start;
