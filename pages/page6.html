Что такое одномерные массивы?

Одномерный массив представляет собой набор элементов одного типа, расположенных последовательно в памяти.

Пример объявления массива:


int[] numbers = new int[5];

Доступ к элементам массива

Для доступа к элементам массива используется индексация. Индексы в C# начинаются с 0.

Пример доступа к элементам массива:


int[] numbers = { 1, 2, 3, 4, 5 };
Console.WriteLine(numbers[0]); // Выводит первый элемент массива
Console.WriteLine(numbers[4]); // Выводит последний элемент массива

Инициализация массива

Массивы могут быть инициализированы при объявлении или позже с использованием цикла или явной инициализации.

Примеры инициализации массива:


int[] numbers = new int[] { 1, 2, 3, 4, 5 }; // Явная инициализация
int[] numbers2 = { 1, 2, 3, 4, 5 }; // Сокращенная инициализация

Примеры программ с массивами

Пример 1: Создание массива и вывод его элементов:


using System;

class Program
{
    static void Main()
    {
        // Создаем массив целых чисел
        int[] numbers = new int[5];

        // Инициализируем массив значениями
        numbers[0] = 10;
        numbers[1] = 20;
        numbers[2] = 30;
        numbers[3] = 40;
        numbers[4] = 50;

        // Выводим элементы массива на экран
        for (int i = 0; i < numbers.Length; i++)
        {
            Console.WriteLine($"Элемент {i}: {numbers[i]}");
        }
    }
}

Пример 2: Использование явной инициализации массива:


using System;

class Program
{
    static void Main()
    {
        // Создаем и инициализируем массив
        int[] numbers = { 10, 20, 30, 40, 50 };

        // Выводим элементы массива на экран
        for (int i = 0; i < numbers.Length; i++)
        {
            Console.WriteLine($"Элемент {i}: {numbers[i]}");
        }
    }
}
