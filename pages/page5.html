Оператор цикла for

Оператор цикла for используется для выполнения блока кода несколько раз.

Пример использования:


for (int i = 0; i < 5; i++)
{
    Console.WriteLine($"Значение переменной i: {i}");
}

Оператор цикла while

Оператор цикла while выполняет блок кода, пока указанное условие истинно.

Пример использования:


int i = 0;
while (i < 5)
{
    Console.WriteLine($"Значение переменной i: {i}");
    i++;
}

Оператор цикла do-while

Оператор цикла do-while также выполняет блок кода, пока указанное условие истинно, но он проверяет условие после каждой итерации цикла.

Пример использования:


int i = 0;
do
{
    Console.WriteLine($"Значение переменной i: {i}");
    i++;
}
while (i < 5);

Применение оператора continue

Оператор continue используется для пропуска текущей итерации цикла и перехода к следующей итерации.

Пример использования:


for (int i = 0; i < 10; i++)
{
    if (i % 2 == 0)
        continue;
    Console.WriteLine($"Нечетное число: {i}");
}

  2. Используя оператор цикла while, выведите на экран таблицу умножения на 5, пропустив результаты умножения на 3.
