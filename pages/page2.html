Операции и выражения

В языке C# операции используются для выполнения различных действий над данными. Они могут быть арифметическими, логическими, сравнениями и другими.

Примеры операций:

  • Арифметические операции: сложение (+), вычитание (-), умножение (*), деление (/), остаток от деления (%).
  • Логические операции: логическое И (&&), логическое ИЛИ (||), логическое отрицание (!).
  • Операции сравнения: равно (==), не равно (!=), больше (>), меньше (<), больше или равно (>=), меньше или равно (<=).

Примеры использования:


int a = 10;
int b = 5;
int sum = a + b; // Сложение
bool result = a > b; // Сравнение

Математические функции (методы)

В C# предоставляется множество математических функций (методов) в классе Math, которые позволяют выполнять различные операции над числами.

Некоторые из наиболее распространенных математических методов включают в себя:

  • Math.Abs(): возвращает абсолютное значение числа.
  • Math.Sqrt(): возвращает квадратный корень числа.
  • Math.Pow(): возвращает указанное число, возведенное в указанную степень.
  • Math.Round(): возвращает число с округлением до ближайшего целого числа.
  • Math.Max(): возвращает наибольшее из двух чисел.
  • Math.Min(): возвращает наименьшее из двух чисел.

Примеры использования:


int num1 = -10;
int absValue = Math.Abs(num1); // absValue равно 10

double number = 16.0;
double squareRoot = Math.Sqrt(number); // squareRoot равно 4.0
