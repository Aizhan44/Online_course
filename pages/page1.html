Основные концепции языка C#

Язык программирования C# (C Sharp) - это современный и мощный язык программирования, разработанный компанией Microsoft. Он широко используется для создания различных типов приложений, включая веб-приложения, настольные приложения, игры и мобильные приложения.

Основные концепции языка C# включают в себя:

  • Синтаксис: форматирование кода, правила написания инструкций.
  • Типы данных: способы представления данных в программе.
  • Переменные: символические имена для хранения данных в памяти.
  • Константы: значения, которые не могут изменяться во время выполнения программы.

Базовый синтаксис

Основы синтаксиса языка C# включают в себя правила для объявления переменных, операторов присваивания и использования комментариев для документирования кода.


int number = 10;
string message = "Hello, world!";
Console.WriteLine(message);

В приведенном выше примере:

  • int - это тип данных для целых чисел.
  • string - это тип данных для строковых значений.
  • Console.WriteLine() - это метод для вывода данных на консоль.

Типы данных и переменные

Тип данных определяет, какие виды данных могут храниться в переменной. В C# существует несколько встроенных типов данных:

  • int: для целых чисел.
  • double: для чисел с плавающей точкой двойной точности.
  • bool: для логических значений (истина или ложь).
  • string: для строковых значений.

Примеры объявления и использования переменных:


int age = 25;
double height = 1.75;
bool isStudent = true;
string name = "John";

Константы

Константы в C# - это именованные значения, которые нельзя изменить во время выполнения программы. Они объявляются с использованием ключевого слова const.


const int DAYS_IN_WEEK = 7;
const string COMPANY_NAME = "ABC Inc.";
