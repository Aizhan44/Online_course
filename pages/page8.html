Введение в двумерные массивы

Двумерный массив - это массив массивов, он представляет собой таблицу из элементов. В отличие от одномерных массивов, где доступ к элементам осуществляется по одному индексу, в двумерных массивах для доступа к элементу требуется указание двух индексов: строки и столбца.

Ввод-вывод элементов двумерного массива

Для ввода-вывода элементов двумерного массива в C# используются вложенные циклы. Внешний цикл перебирает строки, а внутренний - столбцы.

Пример вывода элементов массива:


using System;

class Program
{
    static void Main()
    {
        // Создаем и инициализируем двумерный массив
        int[,] matrix = { { 1, 2, 3 }, { 4, 5, 6 }, { 7, 8, 9 } };

        // Выводим элементы массива на экран
        for (int i = 0; i < matrix.GetLength(0); i++)
        {
            for (int j = 0; j < matrix.GetLength(1); j++)
            {
                Console.Write(matrix[i, j] + " ");
            }
            Console.WriteLine();
        }
    }
}

Пример ввода элементов массива:


using System;

class Program
{
    static void Main()
    {
        // Создаем двумерный массив
        int[,] matrix = new int[3, 3];

        // Заполняем массив данными, введенными пользователем
        for (int i = 0; i < matrix.GetLength(0); i++)
        {
            for (int j = 0; j < matrix.GetLength(1); j++)
            {
                Console.Write($"Введите элемент [{i}, {j}]: ");
                matrix[i, j] = int.Parse(Console.ReadLine());
            }
        }

        // Выводим элементы массива на экран
        for (int i = 0; i < matrix.GetLength(0); i++)
        {
            for (int j = 0; j < matrix.GetLength(1); j++)
            {
                Console.Write(matrix[i, j] + " ");
            }
            Console.WriteLine();
        }
    }
}

Сумма элементов главной и побочной диагонали

Для подсчета сумм элементов главной и побочной диагонали двумерного массива также используются вложенные циклы. Главная диагональ проходит от верхнего левого угла до нижнего правого угла, побочная - от верхнего правого угла до нижнего левого.

Пример:


using System;

class Program
{
    static void Main()
    {
        int[,] matrix = { { 1, 2, 3 }, { 4, 5, 6 }, { 7, 8, 9 } };
        int rows = matrix.GetLength(0);
        int cols = matrix.GetLength(1);

        int mainDiagonalSum = 0;
        int secondaryDiagonalSum = 0;

        for (int i = 0; i < rows; i++)
        {
            for (int j = 0; j < cols; j++)
            {
                if (i == j)
                {
                    mainDiagonalSum += matrix[i, j];
                }

                if (i + j == rows - 1)
                {
                    secondaryDiagonalSum += matrix[i, j];
                }
            }
        }

        Console.WriteLine($"Сумма элементов главной диагонали: {mainDiagonalSum}");
        Console.WriteLine($"Сумма элементов побочной диагонали: {secondaryDiagonalSum}");
    }
}
