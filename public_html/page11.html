Символы и строки в С#

Символы и строки

В C# символы представлены типом char, который представляет отдельный символ Unicode. Строки представлены типом string, который представляет последовательность символов Unicode.

Символы могут быть определены с помощью одинарных кавычек, а строки - с помощью двойных кавычек.

Пример:


char symbol = 'A';
string greeting = "Hello, world!";

Основные методы класса System.Char

Класс System.Char предоставляет различные методы для работы с символами в C#:

  • IsDigit(char) - проверяет, является ли символ цифрой.
  • IsLetter(char) - проверяет, является ли символ буквой.
  • IsWhiteSpace(char) - проверяет, является ли символ пробельным.
  • IsLower(char) - проверяет, является ли символ строчной буквой.
  • IsUpper(char) - проверяет, является ли символ заглавной буквой.
  • ToUpper(char) - преобразует символ в верхний регистр.
  • ToLower(char) - преобразует символ в нижний регистр.

Операции над строками

В C# можно совершать различные операции над строками:

  • Конкатенация строк (используя оператор +).
  • Получение подстроки (метод Substring).
  • Сравнение строк (методы Compare и операторы сравнения).
  • Поиск подстроки (метод IndexOf).
  • Замена подстроки (метод Replace).
  • Разделение строки на подстроки (метод Split).
  • Удаление пробелов (методы Trim, TrimStart, TrimEnd).

Методы класса String

Класс System.String предоставляет множество методов для работы со строками в C#:

  • Length - возвращает длину строки.
  • IndexOf - возвращает индекс первого вхождения указанного символа или подстроки.
  • LastIndexOf - возвращает индекс последнего вхождения указанного символа или подстроки.
  • Substring - возвращает подстроку указанной строки.
  • ToUpper - возвращает строку, в которой все символы приведены к верхнему регистру.
  • ToLower - возвращает строку, в которой все символы приведены к нижнему регистру.
  • Trim - возвращает строку без начальных и конечных пробелов.
  • Replace - заменяет все вхождения указанной строки на другую строку.

Массивы строк

Массив строк (string[]) - это упорядоченная коллекция строк. В C# массивы создаются с помощью ключевого слова new.

Пример:


string[] colors = new string[] { "Red", "Green", "Blue" };

Форматирование данных числовых типов

В C# для форматирования числовых данных используются методы класса ToString и спецификаторы формата. Например:


int number = 123;
string formattedNumber = number.ToString("D5"); // "00123"

В данном примере число 123 форматируется как пятизначное число с ведущими нулями.

Примеры использования методов класса System.Char

Пример 1: Проверка символа на цифру

Давайте проверим символ на то, является ли он цифрой, используя метод IsDigit:


char symbol = '7';
bool isDigit = Char.IsDigit(symbol); // true

Пример 2: Преобразование символа в верхний регистр

Теперь давайте преобразуем символ в верхний регистр с помощью метода ToUpper:


char lowercase = 'a';
char uppercase = Char.ToUpper(lowercase); // 'A'

Пример 3: Проверка символа на пробел

Наконец, давайте проверим символ на то, является ли он пробельным, используя метод IsWhiteSpace:


char space = ' ';
bool isWhitespace = Char.IsWhiteSpace(space); // true

Примеры операций над строками

Пример 1: Конкатенация строк

Склеим две строки вместе:


string firstName = "John";
string lastName = "Doe";
string fullName = firstName + " " + lastName; // "John Doe"

Пример 2: Получение подстроки

Давайте получим подстроку из исходной строки:


string text = "Hello, world!";
string substring = text.Substring(0, 5); // "Hello"

Пример 3: Замена подстроки

Давайте заменим часть строки на другую строку:


string original = "C# is awesome!";
string modified = original.Replace("awesome", "amazing"); // "C# is amazing!"
