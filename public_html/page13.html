Практические задания

Основы C#

  1. Напишите программу, которая выводит на экран приветствие.
  2. Напишите программу, которая запрашивает у пользователя имя и возраст, а затем выводит их на экран.
  3. Напишите программу, которая считывает два числа с клавиатуры и выводит их сумму.
  4. Напишите программу, которая определяет, является ли введенное число четным или нечетным.
  5. Напишите программу, которая находит сумму всех чисел от 1 до 100.

Работа с переменными

  1. Напишите программу, которая запрашивает у пользователя два числа и выводит их сумму.
  2. Создайте программу, которая находит среднее арифметическое нескольких чисел, введенных пользователем.
  3. Напишите программу для перевода температуры из градусов Цельсия в градусы Фаренгейта. Пользователь должен ввести значение в градусах Цельсия, а программа должна вывести соответствующее значение в градусах Фаренгейта. Формула для перевода: F = C * 9/5 + 32
  4. Напишите программу, которая запрашивает у пользователя длину и ширину прямоугольника, вычисляет его площадь и выводит результат на экран.
  5. Напишите программу, которая запрашивает у пользователя два числа, затем меняет их значения местами и выводит результат на экран. Например, если пользователь вводит 5 и 8, программа должна вывести "Первое число: 8, Второе число: 5".

Условные операторы

  1. Напишите программу, которая выводит на экран числа от 1 до 10.
  2. Напишите программу, которая находит сумму всех четных чисел от 1 до 100.
  3. Напишите программу, которая запрашивает у пользователя число и выводит его таблицу умножения.
  4. Напишите программу, которая определяет, является ли введенное число простым.
  5. Напишите программу, которая выводит на экран все делители введенного числа.

    Циклы

    1. Напишите программу, которая выводит на экран таблицу умножения от 1 до 10.
    2. Создайте программу, которая находит сумму всех чисел от 1 до 100.
    3. Напишите программу, которая использует цикл for для вывода чисел от 1 до 10.
    4. Напишите программу, которая запрашивает у пользователя целые числа с клавиатуры и подсчитывает их сумму, используя цикл while. Программа должна завершиться, когда пользователь введет 0.
    5. Напишите программу, которая генерирует последовательность чисел Фибоначчи до определенного предела. Пользователь должен указать этот предел. Последовательность Фибоначчи начинается с 0 и 1, а каждое последующее число равно сумме двух предыдущих чисел.

    Работа с массивами

    1. Сумма элементов массива: Напишите программу, которая находит сумму всех элементов в заданном массиве целых чисел.
    2. Поиск минимального элемента: Напишите программу, которая находит минимальный элемент в заданном массиве целых чисел.
    3. Поиск максимального элемента: Напишите программу, которая находит максимальный элемент в заданном массиве целых чисел.
    4. Поиск среднего значения: Напишите программу, которая находит среднее значение (среднее арифметическое) всех элементов в заданном массиве вещественных чисел.
    5. Проверка наличия элемента: Напишите программу, которая проверяет, содержится ли определенное значение в заданном массиве целых чисел, и выводит соответствующее сообщение.

    Работа с файлами

    1. Чтение из файла: Напишите программу, которая открывает текстовый файл для чтения и выводит его содержимое на экран.
    2. Запись в файл: Создайте программу, которая запрашивает у пользователя строки текста и записывает их в текстовый файл.
    3. Анализ файла: Напишите программу, которая анализирует содержимое текстового файла и подсчитывает количество слов, символов и строк.
    4. Копирование файла: Реализуйте программу, которая копирует содержимое одного файла в другой.
    5. Поиск по файлу: Напишите программу, которая открывает текстовый файл для чтения и запрашивает у пользователя слово для поиска. Затем программа выводит все строки файла, содержащие это слово.
