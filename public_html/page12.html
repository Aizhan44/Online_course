Объектно-ориентированное программирование

Объектно-ориентированное программирование (ООП) - это методология программирования, основанная на использовании объектов, которые являются экземплярами классов.

В C# основные концепции ООП включают в себя классы, объекты, абстракцию, инкапсуляцию, полиморфизм, наследование и другие.

Классы и объекты

Классы являются основными строительными блоками в объектно-ориентированном программировании. Они описывают состояние и поведение объектов.

Пример для класса:

Вот пример класса в C#: 


public class Car
{
    public string Make { get; set; }
    public string Model { get; set; }
    public int Year { get; set; }

    public Car(string make, string model, int year)
    {
        Make = make;
        Model = model;
        Year = year;
    }
}
// Конструктор public Car(string make, string model, int year) { Make = make; Model = model; Year = year; } } 


Объекты создаются на основе классов с помощью оператора new:

Car myCar = new Car("Toyota", "Corolla", 2020);

Пример для объекта:
using System;
public class Person
{
    public string Name;
    public int Age;
    public Person(string name, int age)
    {
        Name = name;
        Age = age;
    }
    public void DisplayInfo()
    {
        Console.WriteLine($"Name: {Name}, Age: {Age}");
    }
}
class Program
{
    static void Main(string[] args)
    {
        Person person1 = new Person("Alice", 30);
        Person person2 = new Person("Bob", 25);
        person1.DisplayInfo();
        person2.DisplayInfo();
    }
}
Пример для абстракции:
using System;
public interface IVehicle
{
    void Drive();
}
public class Car : IVehicle
{
    public void Drive()
    {
        Console.WriteLine("Car is being driven.");
    }
}
class Program
{
    static void Main(string[] args)
    {
        Car car = new Car();
        car.Drive();
    }
}
Пример для наследования:
using System;
public class Animal
{
    public virtual void MakeSound()
    {
        Console.WriteLine("Animal makes a sound");
    }
}
public class Dog : Animal
{
    public override void MakeSound()
    {
        Console.WriteLine("Dog barks");
    }
}
class Program
{
    static void Main(string[] args)
    {
        Animal animal = new Animal();
        Dog dog = new Dog();
        animal.MakeSound();
        dog.MakeSound();
    }
}
Пример для полиморфизма:
public class Animal
{
    public virtual void MakeSound()
    {
        Console.WriteLine("Животное издает звук");
    }
}
public class Dog : Animal
{
    public override void MakeSound()
    {
        Console.WriteLine("Собака лает");
    }
}
public class Cat : Animal
{
    public override void MakeSound()
    {
        Console.WriteLine("Кошка мяучит");
    }
}
class Program
{
    static void Main(string[] args)
    {
        Animal[] animals = new Animal[] { new Dog(), new Cat() };
        foreach (var animal in animals)
        {
            animal.MakeSound();
        }
    }
}

Пример для инкапсуляции:
public class Car
{
    private string make;
    private string model;
    private int year;
    public string Make
    {
        get { return make; }
        set { make = value; }
    }
    public string Model
    {
        get { return model; }
        set { model = value; }
    }
    public int Year
    {
        get { return year; }
        set { year = value; }
    }
}
class Program
{
    static void Main(string[] args)
    {
        Car myCar = new Car();
        myCar.Make = "Toyota";
        myCar.Model = "Corolla";
        myCar.Year = 2020;
        Console.WriteLine($"Марка: {myCar.Make}, Модель: {myCar.Model}, Год: {myCar.Year}");
    }
}

Абстракция


Абстракция - это процесс сокрытия деталей реализации и показа только 
необходимых деталей.


В C# абстракцию можно достичь с помощью классов и интерфейсов.


Задание 2:


Создайте интерфейс IVehicle, который определяет метод 
Drive. Затем создайте класс Car, который реализует 
этот интерфейс.


Инкапсуляция


Инкапсуляция - это механизм, который ограничивает доступ к данным внутри 
объекта и скрывает их от внешнего мира.


В C# инкапсуляцию можно достичь с помощью модификаторов доступа, таких как 
private, protected, public.


Задание 3:


Создайте класс BankAccount, который имеет приватное поле 
balance и методы Deposit и Withdraw, 
которые изменяют этот баланс.


Полиморфизм


Полиморфизм - это возможность объекта обрабатывать данные различными 
способами в зависимости от их типа или класса.


В C# полиморфизм реализуется через перегрузку методов, виртуальные методы и 
интерфейсы.


Задание 4:


Создайте базовый класс Shape и производные классы 
Circle и Rectangle. У каждого класса должен быть метод 
CalculateArea, который возвращает площадь 
фигуры.


Наследование


Наследование - это механизм, позволяющий создавать новый класс на основе 
существующего класса.


В C# классы могут наследовать другие классы с помощью ключевого слова 
class.


Задание 5:


Создайте класс Animal, а затем производные классы 
Dog и Cat. Каждый класс должен иметь метод 
MakeSound.


Оператор new


Оператор new используется для создания новых объектов на основе 
классов.


Пример:

Car myCar = new Car();
            


Конструкторы


Конструкторы - это специальные методы, вызываемые при создании нового 
объекта.


В C# конструкторы имеют тот же название, что и класс, и не возвращают 
значение.

public class Car
{
    public string Make;
    public string Model;
    public int Year;

    // Конструктор
    public Car(string make, string model, int year)
    {
        Make = make;
        Model = model;
        Year = year;
    }
}
