C# тілінде қандай функциялар мен әдістер деп нені айтамыз?
C# тілінде функциялар мен әдістер бірдей.
Функциялар - бұл бағдарламаның басқа бөліктерінен шақыруға болатын белгілі бір тапсырманы орындайтын код бөліктері, ал әдістер класс ішінде анықталған функциялар.
Әдістер - бұл бағдарламаның басқа бөліктерінен шақыруға болатын белгілі бір тапсырманы орындайтын код бөліктері, ал функциялар сынып ішінде анықталған функциялар.
C# тіліндегі операторлар дегеніміз не?
Операторлар айнымалыларды жариялау үшін қолданылатын түйінді сөздер.
Операторлар – деректермен операцияларды орындау үшін қолданылатын символдар немесе түйінді сөздер.
Операторлар - бұл арнайы тапсырмаларды орындау үшін шақырылған C# тіліндегі арнайы функциялар.
C# тілі қандай арифметикалық операторларды қолдайды?
+, -, *, /
&&, ||
==, !=
C# тілінде логикалық операцияларды орындау үшін қандай логикалық операторлар қолданылады?
&&, ||
+, -, *
==, !=
C# тіліндегі шартты мәлімдемелер дегеніміз не?
Шартты операторлар – мәліметтер типін анықтау үшін қолданылатын белгілер.
Шартты операторлар – шартқа байланысты белгілі бір код блогын орындауға мүмкіндік беретін конструкциялар.
Шартты операторлар жаңа айнымалылар құруға арналған түйінді сөздер болып табылады.
C# тілінде циклдердің қандай түрлеріне қолдау көрсетіледі?
for, if
while, do-while
switch, case
C# тілінде циклдарды басқару үшін қандай кілт сөздер қолданылады?
break, continue
switch, case
if, else
C# тіліндегі while және do-while циклдерінің айырмашылығы неде?
while циклі кемінде бір рет орындалады, бірақ do-while циклі бір рет болса да орындалмауы мүмкін.
do-while циклі кемінде бір рет орындалады, бірақ while циклі бір рет болса да орындалмауы мүмкін.
Олардың арасында ешқандай айырмашылық жоқ.
C# тілінде функциялар мен әдістерді жариялау үшін қандай кілт сөздер қолданылады?
func, method
def, function
void, return
Функция және әдіс параметрлері дегеніміз не?
Параметрлер – функцияға немесе өңдеу әдісіне берілетін арнайы айнымалылар.
Параметрлер функция немесе әдіс ішінде анықталған және деректерді сақтау үшін қолданылатын айнымалылар.
Параметрлер функция немесе әдіс шақырылатын және өңдеу үшін оған жіберілетін нүктеде анықталатын арнайы айнымалылар.
C# тіліндегі массивтер дегеніміз не?
Массив – реттілікпен орналасқан, индекс арқылы қатынасатын бір типті элементтердің жиыны.
Массив – бір немесе бірнеше элементтерден тұратын айнымалы.
Массив - бұл әр элементтің өз индексі болатын кесте болып табылатын деректер құрылымы.
C# тілінде массивті қалай жариялауға және инициализациялауға болады?
int массиві[] = new int[5] {1, 2, 3, 4, 5};
int[] массиві = new int[5] {1, 2, 3, 4, 5};
int массиві[5] = {1, 2, 3, 4, 5};
C# тілінде индекс бойынша массив элементтеріне қалай қол жеткізуге болады?
Индекстер 1-ден басталады.
индекстер 0-ден басталады.
индекстер -1-ден басталады.
C# тілінде ерекше жағдайды қалай өңдеуге болады?
Қатені тексеру үшін if-else операторларын пайдаланыңыз.
Ерекше жағдайды ұстау және өңдеу үшін try-catch пайдаланыңыз.
Сәйкес ерекше жағдайды өңдеуді таңдау үшін switch-case операторын пайдаланыңыз.
OOP (объектіге бағытталған бағдарламалау) дегеніміз не?
OOP - бұл сыныптардың даналары болып табылатын объектілерді пайдалануға негізделген бағдарламалау әдістемесі.
OOP - бұл объектілермен жұмыс істеу үшін арнайы жасалған бағдарламалау тілі.
OOP – сыныптар мен объектілерді пайдаланбайтын бағдарламалаудың архитектуралық стилі.
