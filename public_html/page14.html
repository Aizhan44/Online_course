Асинхронды программалау

C# бағдарламалау курсының он төртінші сабағына қош келдіңіздер! Бұл оқулықта біз асинхронды бағдарламалауды және C# бағдарламалау тілінде асинхронды әдістерді пайдалануды үйренеміз.

Асинхронды  программалау

C# тілінде асинхронды бағдарламалау желіден деректерді жүктеу немесе файлдан оқу сияқты енгізу/шығару операцияларын орындау кезінде жұмысын жалғастыратын қолданбаларды жасауға мүмкіндік береді. Бұл қолданбаның жауап беру қабілеті мен өнімділігін жақсартады.

Мысал

C# тілінде асинхронды бағдарламалауды қолданудың мысалын қарастырайық:


using System;
using System.Net.Http;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        await DownloadDataAsync();
    }

    static async Task DownloadDataAsync()
    {
        using (HttpClient client = new HttpClient())
        {
            string url = "https://api.example.com/data";
            string data = await client.GetStringAsync(url);
            Console.WriteLine("Деректер алынды:");
            Console.WriteLine(data);
        }
    }
}
