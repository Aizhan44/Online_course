Мұрагерлік және полиморфизм

C # бағдарламалау курсының тоғызыншы сабағына қош келдіңіз! Бұл сабақта біз C # бағдарламалау тіліндегі мұрагерлік пен полиморфизмді зерттейміз.

Мұрагерлік

Мұрагерлік бұрыннан бар негізінде жаңа класс жасауға мүмкіндік береді. Басқа классты мұрагер ететін класс туынды класс деп аталады, ал мұрагер класс базалық класс деп аталады.

Мысал

C# тіліндегі мұраның мысалын қарастырайық:


using System;

// Базовый класс
class Animal
{
    public void Eat()
    {
        Console.WriteLine("Жануарлар тамақтанды.");
    }
}

// Производный класс
class Dog : Animal
{
    public void Bark()
    {
        Console.WriteLine("Ит үреді.");
    }
}

class Program
{
    static void Main()
    {
        Dog myDog = new Dog();
        myDog.Eat(); // Негізгі класс әдісі
        myDog.Bark(); // Туынды кластан алынған әдіс
    }
}
