Глосарий

Определения

C#
Язык программирования, разработанный Microsoft, основанный на языке C++, предназначенный для разработки приложений под платформу .NET.
.NET Framework
Фреймворк разработки программного обеспечения, созданный Microsoft, предназначенный для разработки и выполнения программного обеспечения, написанного на различных языках программирования.
Переменная
Именованное хранилище для данных в программе, которое может быть изменено в процессе выполнения программы.
Условный оператор
Конструкция в программировании, позволяющая выполнять различные блоки кода в зависимости от условий.
Цикл
Конструкция в программировании, которая позволяет многократно выполнять определенный блок кода.
Метод
Функция или процедура, связанная с объектом или классом в объектно-ориентированном программировании, которая выполняет определенное действие.
Интерфейс
В объектно-ориентированном программировании - контракт, определяющий набор методов, которые класс должен реализовать. В GUI - средство взаимодействия пользователя с программой.
Полиморфизм
Концепция в объектно-ориентированном программировании, позволяющая объектам одного и того же типа реагировать на вызовы методов по-разному.
Наследование
Механизм в объектно-ориентированном программировании, позволяющий создавать новые классы на основе существующих, заимствуя их свойства и методы.
Исключение
Событие, возникающее в процессе выполнения программы, указывающее на ошибку или непредвиденную ситуацию, которая требует особого обращения.

Аббревиатуры

IDE
Среда разработки (Integrated Development Environment) - программное обеспечение, которое объединяет в себе инструменты для разработки, отладки и тестирования программного обеспечения.
API
Интерфейс прикладного программирования (Application Programming Interface) - набор готовых классов, процедур, функций и структур, предоставляемых библиотекой или сервисом для использования во внешних программных продуктах.
GUI
Графический пользовательский интерфейс (Graphical User Interface) - способ представления информации и взаимодействия пользователя с программой с помощью графических элементов, таких как кнопки, текстовые поля и окна.
OOP
Объектно-ориентированное программирование (Object-Oriented Programming) - методология программирования, основанная на концепции объектов, которые могут содержать данные в виде полей и код в виде методов.
CLR
Общеязыковая исполняющая среда (Common Language Runtime) - виртуальная машина, предоставляемая платформой .NET, которая выполняет IL-код (Intermediate Language), компилированный из программы на C# или других языках .NET.
SQL
Structured Query Language - язык структурированных запросов, используемый для взаимодействия с базами данных.
JSON
JavaScript Object Notation - формат обмена данными, основанный на синтаксисе JavaScript, часто используемый для передачи данных между клиентом и сервером в веб-приложениях.
XML
Extensible Markup Language - расширяемый язык разметки, используемый для хранения и передачи структурированных данных в формате, пригодном для чтения как людьми, так и компьютерами.
HTML
HyperText Markup Language - язык разметки, используемый для создания веб-страниц и их структурирования.
CSS
Cascading Style Sheets - язык таблиц стилей, используемый для оформления элементов HTML и управления их внешним видом.
