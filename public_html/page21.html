Microsoft Visual Studio

Знакомство со средой Microsoft Visual Studio (язык C#). Microsoft Visual Studio с поддержкой C# - мощный инструмент для разработки программного обеспечения на языке C#. Microsoft создала удобную среду разработки (IDE) для своего языка - Visual Studio. Вам не нужно устанавливать десятки SDK, настраивать текстовые редакторы и многое другое, чтобы начать разработку. Visual Studio включает в себя все необходимые инструменты. Все, что вам нужно сделать, это установить его, и вы можете легко делать все, что захотите.

Вы можете скачать программу с сайта, представленного ниже.

https://visualstudio.microsoft.com/ru/downloads/
